# Projetos em Python

Bem-vindo ao repositório que abrange diversos projetos e recursos relacionados à programação em Python. Aqui, você encontrará exemplos de projetos que demonstram a aplicação da linguagem de programação Python em diferentes contextos.

## Projetos

### Projeto 1: Chatbot para Salgadaria

Neste projeto, desenvolvemos um chatbot em Python utilizando bibliotecas como NLTK e ChatterBot. O chatbot é projetado para interagir com clientes de uma salgadaria, fornecendo informações sobre produtos, horários e fazendo pedidos simulados.

- Tecnologias: Python
- Pasta do Projeto: `/projeto_chatbot_salgadaria`

### Projeto 2: Gerador de Senhas

Este projeto consiste em um gerador de senhas seguro em Python. Utilizamos princípios de criptografia e geração aleatória para criar senhas fortes e personalizadas, visando melhorar a segurança online.

- Tecnologias: Python
- Pasta do Projeto: `/projeto_gerador_senhas`
